Getting Started

Basix is a simple no bullsh#t framework for creating responsive prototypes and websites quickly and easily. Built with Sass, Basix makes it effortless for web workers to create multi-device projects with SCSS or CSS.

Intended for web workers that don't need all of the bells, whistles, and dependancies that come with other frameworks, Basix let's you quickly get a responsive framework in place for a web project or prototype without anything more than a handful of pre-determined classes.

The Grid

The Basix Grid alows you to quickly create 12 column nestable grids with only a few classes. Jump right in!

The following are examples of ways to use the Basix Grid. The box-sizing: border-box proptery so that borders and padding do not affect the width of the columns.
